# NHANES-2013-2018-Adult-Asthma
Code and data for analysis of NHANES data, adults with asthma exposed to gas cooking
Project title:
“Impact of recent gas stove use on asthma exacerbations and emergency department visits among adults with asthma”

Project contributors/authors:
Curtis L. Nordgaard, Andee Krasner, Aline Maybank, Flannery Black-Ingersoll, Molly Kile, Jonathan Buonocore, Mary Willis

Markdown R code and accompanying data file can be used to replicate the analysis and data for the manuscript.

"NHANES Gas cooking adult.RMD" contains main analyses and sensitivity analyses. "NHANES Gas cooking adult - unadjusted.RMD" contains same code but with multivariate models replaced by univariate models.

Code was built and revised by Curtis Nordgaard, Aline Maybank, and Flannery Black-Ingersoll.
